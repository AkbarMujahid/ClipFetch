# 🎬 ClipFetch

ClipFetch is a simple and powerful video downloader web app that allows users to download videos from platforms like YouTube, Instagram, and Facebook in different qualities.

Built with a clean UI and a Node.js backend using yt-dlp, it provides a smooth and fast downloading experience.

---

## 🚀 Features

* 🎥 Download videos in multiple qualities
* 🌐 Supports YouTube, Instagram, Facebook links
* ⚡ Fast and lightweight backend using yt-dlp
* 🎨 Modern UI with animations and dark/light mode
* 🖼 Thumbnail preview before download
* 📜 Download history with re-download option
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠 Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js (Express)
* Video Processing: yt-dlp (Python)
* Styling: Custom CSS + Font Awesome

## ⚙️ Installation (Local Setup)

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR_USERNAME/ClipFetch.git
cd ClipFetch
```

---

### 2️⃣ Install backend dependencies

```
cd server
npm install
```

---

### 3️⃣ Install Python dependencies

Make sure Python is installed, then run:

```
pip install yt-dlp
```

---

### 4️⃣ Install FFmpeg

Download and install FFmpeg and add it to your system PATH.

---

### 5️⃣ Run the server

```
node server.js
```

---

### 6️⃣ Open in browser

```
http://localhost:3000
```

---

## 📸 Usage

1. Paste a video URL
2. Click **Get Formats**
3. Choose desired quality
4. Download starts instantly

---

## ⚠️ Notes

* This project works best in a local environment
* Some hosting platforms (like Render, Vercel) may restrict yt-dlp
* For full functionality online, use a VPS
